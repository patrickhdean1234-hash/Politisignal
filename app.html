🌐 Data Sources

PolitiSignal pulls from these free sources, updated every 5 minutes via automated fetcher.

●.gov RSS Feeds
Official press releases from senate.gov and house.gov — free, official, highly reliable.
●Bluesky AT Protocol
Live posts from Pelosi, AOC, Warren, Sanders, Ro Khanna — free public API, no auth needed.
●White House + GovTrack + Fed Register
Official government news, live congressional votes, and new federal regulations.
●SEC EDGAR Form 4
Congressional stock trades and insider filings — live from SEC's public ATOM feed.
●USTR + Politico + The Hill
Trade Representative press releases, Politico politics feed, The Hill headlines — all free RSS.
●Reddit (r/politics, r/investing, r/wallstreetbets)
Top market-relevant posts via Reddit JSON API — free, no auth required.
●Truth Social
Trump's public posts via Mastodon API — free, no auth required.
●YouTube Data API
Official channel videos — free 10k units/day. Requires YOUTUBE_API_KEY in GitHub Secrets.
●Twitter/X
API now costs $100/month. Not connected — Bluesky and RSS give us better coverage for free.
